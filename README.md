# Leg8t Empanada Ordering System

## Final Project

Leg8t Empanada is a web-based ordering system developed as a Final Project for **CCS0043**.

## Features

### Buyer

- User Registration
- Email Confirmation
- Login
- Logout
- Product Store
- Shopping Cart
- Checkout
- Payment
- Order History

### Seller

- Dashboard
- Product Management
- User Management
- Inventory Management
- Orders Management
- Audit Log
